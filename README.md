# Realizei um mini projeto da Calculadora Básica, utilizando HTML, CSS e JavaScript.
Calculadora com operadores simples, como: Divisão, Multiplicação, Subtração e Adição (realizado no HMTL)
Estilo da Calculadora foi feito no CSS, principal diferencial: box-shadow.
JavaScript onde foi definido o que cada operação faz, tanto o sinal de "=" e "AC".
